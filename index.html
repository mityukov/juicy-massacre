Сочная резня! Карвинг - резка по фруктам, овощам

Download this project as a .zip file Download this project as a tar.gz file

Карвинг - искусство резки по овощам и фруктам, пришедшее к нам из Азии и в частности Таиланда. Он уже стал модным в Европе и в последнее десятилетие завоевывает популярность в России.

Вы хотите удивить семью, друзей и знакомых оригинальным оформлением стола? Или хотите сделать необычный подарок? Если да, то Вам сюда! Композиция из овощей или вкусный букет подарит много ярких впечатлений и поможет сделать событие незабываемым.
